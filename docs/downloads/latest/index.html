Upgrading from v1.4.0? #

If you get the database error, “MMEX database version 13 doesn’t work with this MMEX version,” then see Steps to convert your database from 1.4.x

The latest Money Manager Ex release uses the database structure of the 1.3.x series and discontinued the structure of version 1.4.x. This maintains compatibility with the mobile app.
